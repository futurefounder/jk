April 28, 2021

Website builders: There are now so many like a dime a dozen.

When I asked myself how to set up this website, I thought pretty quickly:

“Definitely not with Wordpress”

Don’t get me wrong, Wordpress is a great solution in some cases.

Just not for me.

I wanted full flexibility with the least possible effort in the creation, as in the maintenance, and that’s why I ended up with Hugo.

What is Hugo?#

Hugo is an open source website generator.

You can use it to create very fast websites very quickly.

It’s fun again because it’s easy.

In addition, I can concentrate again on what is important to me:

Create content regularly.
